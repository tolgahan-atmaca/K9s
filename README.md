## k9s Offline Installation

This document describes how to install **k9s** on an **air-gapped (offline) Linux server** using a `.deb` package.

The installation is performed by:
- Downloading the package on a machine with internet access
- Transferring it through a Windows jumpbox
- Installing it on the target Linux server without internet access

This guide is intended for **DevOps / SRE teams** working in restricted network environments.

---

## k9s Offline Kurulum

Bu doküman, **internete kapalı (air-gapped) bir Linux sunucuya** `.deb` paketi kullanarak **k9s** kurulumu nasıl yapılır, onu anlatır.

Kurulum süreci şu adımlardan oluşur:
- İnternet erişimi olan bir bilgisayarda paketin indirilmesi
- Windows jumpbox üzerinden hedef sunucuya aktarılması
- İnternetsiz Linux sunucuda kurulumun tamamlanması

Bu rehber, **kısıtlı ağ ortamlarında çalışan DevOps / SRE ekipleri** için hazırlanmıştır.
